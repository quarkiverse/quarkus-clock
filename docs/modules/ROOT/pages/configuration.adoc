Clock behavior is controlled by the `quarkus.clock` configuration namespace.

== Clock types

[cols="1,3"]
|===
|Value |Behavior

|`system`
|Default. Uses `Clock.systemUTC()`.

|`fixed`
|Returns a constant instant. Requires `quarkus.clock.fixed-instant`.

|`adjustable`
|Provides a mutable `AdjustableClock` that can be changed at runtime.
|===

Configuration values are case-insensitive (`system`, `fixed`, and `adjustable` are accepted).
